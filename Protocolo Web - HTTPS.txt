Capítulo. Protocolo Web - HTTPS.


O HTTPS (Hypertext Transfer Protocol Secure) é uma extensão do protocolo HTTP (Hypertext Transfer Protocol) que adiciona uma camada de segurança por meio do uso de criptografia. O HTTPS é usado para proteger a integridade e a confidencialidade dos dados transmitidos entre um cliente e um servidor na Web.

Aqui estão algumas características e conceitos importantes relacionados ao HTTPS:

Item. 1. Criptografia SSL/TLS: O HTTPS utiliza protocolos de criptografia SSL (Secure Sockets Layer) ou TLS (Transport Layer Security) para proteger os dados durante a comunicação. Isso garante que as informações transmitidas entre o cliente e o servidor sejam criptografadas e não possam ser interceptadas ou alteradas por terceiros.

Item. 2. Certificados SSL/TLS: Para estabelecer uma conexão HTTPS, um servidor precisa ter um certificado SSL/TLS válido emitido por uma autoridade de certificação confiável. O certificado é usado para autenticar o servidor e criptografar a comunicação.

Item. 3. Porta padrão: A porta padrão para o HTTPS é a 443. Quando um cliente acessa um site ou serviço usando HTTPS, ele se conecta à porta 443 do servidor.

Item. 4. Handshake SSL/TLS: Antes de estabelecer uma conexão segura, o cliente e o servidor passam por um processo chamado handshake SSL/TLS. Durante esse processo, eles trocam informações de segurança, autenticam-se mutuamente e acordam um conjunto de algoritmos criptográficos a serem usados para a comunicação.

Item. 5. Indicador de segurança: Os navegadores modernos exibem um indicador visual para indicar que uma conexão HTTPS está em uso. Isso geralmente é representado por um cadeado fechado ou um ícone de cadeado verde na barra de endereço do navegador. Além disso, os URLs começam com "https://" em vez de "http://".

Item. 6. Benefícios de segurança: O uso do HTTPS oferece vários benefícios de segurança, como a proteção contra ataques de interceptação de dados, autenticação do servidor, integridade dos dados transmitidos e confidencialidade das informações.

Item. 7. SEO e confiança do usuário: O HTTPS se tornou um fator importante para mecanismos de busca, como o Google, e também para a confiança dos usuários. Ter um site configurado com HTTPS pode melhorar a classificação nos resultados de pesquisa e transmitir confiança aos visitantes do site.

O HTTPS é amplamente adotado para proteger a privacidade e a segurança dos usuários durante a comunicação na Web. É especialmente essencial para transações financeiras, envio de dados sensíveis e qualquer troca de informações confidenciais.
