Capítulo. Mensageria, stream e CORBA.


Mensageria, stream e CORBA são tecnologias e padrões relacionados à comunicação entre sistemas distribuídos. Cada um possui características e casos de uso específicos. Vamos abordar cada um deles separadamente:

Item. 1. Mensageria: A mensageria é uma abordagem que permite a comunicação assíncrona entre sistemas por meio da troca de mensagens. Nesse modelo, os sistemas enviam e recebem mensagens em filas ou tópicos, sem a necessidade de uma interação síncrona direta. Isso oferece alta flexibilidade, escalabilidade e desacoplamento entre os sistemas. Exemplos de tecnologias de mensageria incluem o RabbitMQ, ActiveMQ e Apache Kafka.

Item. 2. Stream: Streaming é uma abordagem de processamento de dados em tempo real, onde os dados são transmitidos continuamente e processados à medida que chegam. É comumente utilizado em cenários que envolvem grandes volumes de dados e necessitam de análises em tempo real. As tecnologias de streaming populares incluem o Apache Kafka, Apache Flink e Apache Spark Streaming.

Item. 3. CORBA: O CORBA (Common Object Request Broker Architecture) é um padrão para comunicação entre objetos distribuídos em sistemas heterogêneos. Ele define uma infraestrutura de middleware que permite que objetos em diferentes linguagens de programação e plataformas se comuniquem de forma transparente. O CORBA utiliza uma arquitetura baseada em solicitação e resposta, onde os objetos invocam operações remotas uns nos outros por meio de um Object Request Broker (ORB). Embora tenha sido amplamente utilizado no passado, o CORBA foi substituído por abordagens mais modernas, como serviços web e REST.

Em resumo, a mensageria é uma forma de comunicação assíncrona baseada na troca de mensagens, o streaming é utilizado para processamento contínuo de dados em tempo real e o CORBA é um padrão para comunicação entre objetos distribuídos em sistemas heterogêneos. Cada um deles tem suas características específicas e pode ser aplicado em diferentes contextos e requisitos de comunicação entre sistemas distribuídos.
